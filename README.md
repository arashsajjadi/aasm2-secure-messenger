# AASM2 Secure Messenger

Cross-platform secure messaging and file encryption project with a shared AASM2 format.

## Projects
- `android_project/` Android app
- `desktop_project/` Desktop app for Linux and Windows
- `spec/` Shared format specification and compatibility notes

## Current status
Initial monorepo with Android and desktop sources.

## Goals
- Shared encrypted format across platforms
- Android APK builds
- Linux desktop builds
- Windows desktop builds
- Future compatibility improvements and release packaging
