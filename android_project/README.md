# AASM2 Android

Native Android project built with Kotlin and Jetpack Compose.

## Highlights

- Text encryption and decryption
- File encryption and decryption using Android Storage Access Framework
- Compatible with the Desktop project in this bundle
- Material 3 UI
- AES 256 GCM authenticated encryption

## Open in Android Studio

1. Open Android Studio.
2. Choose `Open`.
3. Select the `android_project` folder.
4. Let Gradle sync.

## Build debug APK

Inside Android Studio:

- `Build`
- `Build Bundle(s) / APK(s)`
- `Build APK(s)`

The generated APK is typically placed under:

```text
app/build/outputs/apk/debug/app-debug.apk
```

## Build release APK

1. Create a signing key if you do not already have one.
2. In Android Studio choose:
   - `Build`
   - `Generate Signed Bundle / APK`
3. Select `APK`
4. Choose your keystore and release options
5. Build

The release APK is typically placed under:

```text
app/build/outputs/apk/release/app-release.apk
```

## Build App Bundle

In Android Studio choose:

- `Build`
- `Generate Signed Bundle / APK`
- `Android App Bundle`

## Command line example

If Gradle wrapper is present on your machine or generated by Android Studio:

```bash
./gradlew assembleDebug
./gradlew assembleRelease
```

On Windows:

```powershell
.\gradlew.bat assembleDebug
.\gradlew.bat assembleRelease
```

## Android version support

- Minimum SDK: 26
- Target SDK: 35
